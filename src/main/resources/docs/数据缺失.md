## 用户未找到

处理这个问题时，直接查询表aaa.aaa中是否存在指定的用户，
如果不存在直接告诉用户数据库中缺失该用户

表结构如下：

```sql
create table aaa.aaa(
    id int,
    name varchar(32)
)
```

## 产品未找到

处理这个问题时，直接查询表aaa.product中是否存在指定的产品，
如果不存在直接告诉用户数据库中缺失该用户
表结构如下：
```sql
create table product(
    id int,
    product_name varchar(32),
    price int
)
```
